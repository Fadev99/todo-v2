# Todos App Version2

## Project setup
Use: ```NodeJS & VueJS```

Version Node: ```v18.15.0```

Client
```
cd client
yarn install
```
Server
```
cd server
yarn install
```

### Compiles and hot-reloads for development
Client
```
yarn serve
```
Server
```
yarn start
```

### Compiles and minifies for production
```
yarn build
```
